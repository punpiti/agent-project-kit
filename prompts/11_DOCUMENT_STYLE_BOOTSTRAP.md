# Prompt: Bootstrap Document Style

Use this once per document-heavy project.

```text
อ่าน .ai/computing-environment/DOCUMENT_PRODUCTION_POLICY.md และ bootstrap เอกสารของโปรเจคนี้

ให้ตรวจว่า project มีสิ่งต่อไปนี้หรือยัง:
- docs/source/
- docs/styles/
- docs/output/ หรือ output/
- .ai/DOCUMENT_PIPELINE.md
- .ai/DOCUMENT_STYLE.md
- .ai/DOCUMENT_QA.md

ถ้ายังไม่มี ให้สร้าง template ที่เหมาะกับ project นี้
แล้วเสนอ style profile หนึ่งชุด:
- document type: formal/public/teaching/internal
- paper/screen target: A4 หรือ 16:9
- Thai font
- English font/fallback
- color palette แบบ minimal
- table style
- build command ที่ควรใช้

อย่าพยายามสร้าง PDF จนกว่า Markdown source จะพร้อมพอ
```
